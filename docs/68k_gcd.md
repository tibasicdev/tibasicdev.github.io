# The gcd() Command

The gcd( command is used, as its name implies, to find the greatest common divisor of two numbers.

```
:gcd(351,65)
              returns 13
```

## Error Conditions

**[260 - Domain error](68k:errors.html#e260)** happens when an argument is not real.
