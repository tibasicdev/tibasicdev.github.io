# Graph Format Tokens
The token 0x7E signifies the beginning of a two-byte graph format token. By and large, these are the settings from the [graph format menu](settings.html) (2nd FORMAT), with some from the mode menu (2nd MODE) thrown in for good measure.

All token values here (and elsewhere in this guide) are in hexadecimal.

- **7E 00** — [Sequential](sequential.html)
- **7E 01** — [Simul](simul.html)
- **7E 02** — [PolarGC](polargc.html)
- **7E 03** — [RectGC](rectgc.html)
- **7E 04** — [CoordOn](coordon.html)
- **7E 05** — [CoordOff](coordoff.html)
- **7E 06** — [Connected](connected.html)
- **7E 07** — [Dot](dot.html)
- **7E 08** — [AxesOn](axeson.html)
- **7E 09** — [AxesOff](axesoff.html)
- **7E 0A** — [GridOn](gridon.html)
- **7E 0B** — [GridOff](gridoff.html)
- **7E 0C** — [LabelOn](labelon.html)
- **7E 0D** — [LabelOff](labeloff.html)
- **7E 0E** — [Web](web.html)
- **7E 0F** — [Time](time.html)

- **7E 10** — [uvAxes](uvaxes.html)
- **7E 11** — [vwAxes](vwaxes.html)
- **7E 12** — [uwAxes](uwaxes.html)
