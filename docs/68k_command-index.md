# 68k Command Index
Some commands have a superscript next to them that indicates compatibility:
- <sup>2.xx</sup> indicates that the command requires AMS 2.xx or higher on a TI-89, TI-92, or TI-92+ (or any 3.00+ version, on the TI-89 Titanium and V200).
- <sup>3.xx</sup> indicates that the command requires a TI-89 Titanium or V200 calculator, as well as AMS 3.xx or higher.
- <sup>Flash</sup> indicates that the command requires a calculator with Flash ROM (that is, it won't work on a TI-92).
