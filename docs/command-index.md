**This page lists all of the commands for the TI-84 Plus C Silver Edition and the TI-84 Plus CE. If you have an older TI-83/84+ with a monochrome display, please visit the [Monochrome Command Index](command-index-bw.html).**

To access more information about the command, click on the command to go to the dedicated page for that command. To view the commands divided into sections, go to the [Command Overview](commands.html). To view most of the commands as they would on the calculator itself through the menus, go to the [Command Menu Map](command-map.html).


Some commands have a superscript next to them that indicates compatibility:
- <sup>CSE</sup> indicates that the command requires a TI-84 Plus CSE
- <sup>CE</sup> indicates that the command requires a TI-84 Plus CE or a specific OS version

See [Compatibility](compatibility.html) for a chart of only the commands that require certain calculators or operating systems.
