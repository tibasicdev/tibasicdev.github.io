# Polar (disambiguation)
Polar is a [graphing mode](graphing-mode.html#polar) on TI-83 series calculators.

It may also refer to:

- [Polar](polar-mode.html), the mode command that enables Polar graphing mode.
- [►Polar](polar-display.html), the format command that displays complex numbers in polar form.
- [PolarGC](polargc.html), the mode command to store graph coordinates as (R,θ).
