The {{getTime}} command returns the current time that the clock has on the TI-84+/SE/CE calculators in [[[lists|list]]] format --  {//hour//, //minute//, //second//}. You can [[[store]]] this list to a variable for later use, or manipulate it the same way you do with other lists. Of course, this command only works if the time has actually been set, so you should use the {{[[[setTime(]]]}} command before using it.

An interesting note about this command is that you cannot index individual elements directly; if you try, each element of the clock is multiplied by the index. You can, however, call the demand and thus store the result in {{[[[Ans]]]}}, and then retrieve the individual elements.
