The {{gcd(}} command returns the [https://en.wikipedia.org/wiki/Greatest_common_divisor greatest common divisor] (GCD) of two nonnegative integers. It also works on lists.
[[code]]
gcd(8,6)
     2
gcd({9,12},6)
     {3 6}
gcd({14,12},{6,8})
     {2 4}
[[/code]]

+ Advanced Uses

A {{gcd(}} command can be nested inside another {{gcd(}} command to compare up to four numbers.

+ Error Conditions

* **[[[errors#dimmismatch|ERR:DIM MISMATCH]]]** is thrown if the arguments are two lists that don't have the same number of elements.
* **[[[errors#domain|ERR:DOMAIN]]]** is thrown if the arguments aren't positive integers (or lists of positive integers) less than {{1E12}}.
