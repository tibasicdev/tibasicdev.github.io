> This table lists all the possible error messages that can occur. The error number to the left is not displayed, but is stored to the [[[nspire:system-variables#errornum|errornum]]] system variable, which can be used in a [[[nspire:Try]]]..[[[nspire:EndTry]]] block. Some error messages are very explicit (like [[[nspire:errors#680|#680]]]), but others are less so. Where the error message itself gave too little information, this table explains the error more thoroughly.

> A large part of the information here was taken from the Nspire CX Reference Guide at [*https://education.ti.com/html/webhelp/EG_TINspire/EN/Subsystems/TI-Nspire%E2%84%A2%20RefeGuide/content/eg_refguide/m_refguide/rg_error_codes_and_messages.htm education.ti.com]
