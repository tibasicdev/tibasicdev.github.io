* Error Conditions
 * [[[inc:68k-error]]]
  * num = error number (e.g. "260")
  * error = error description (e.g. "Domain Error")
  * cause = error cause (e.g. "the argument is not in the range 1-99")

+ Miscellaneous Includes
* [[[inc:Tablet]]] for the programs in the [[[archives]]]
 * [[[inc:tablet-help|Tablet HELP]]] for instructions on how to use the infobox.
 * You probably don't need to worry about the [[[inc:tablet-backend|Backend]]].
* [[[inc:math]]] or [[[inc:68k-math]]] for math formulas in the right background color.
 * eqn = LaTeX equation
* [[[inc:markup]]] or [[[inc:68k-markup]]] for a markup limitations warning.
 * cmd = command to issue a warning about
* [[[inc:next-prev]]] for linking to next/previous pages of a tutorial.
 * prevpage = linkable name of previous page
 * prevtitle = title of previous page
 * tocpage = linkable name of table of contents page
 * toctitle = title of table of contents
 * nextpage = linkable name of next page
 * nexttitle = title of next page
* [[[inc:stub]]] for a page with minimal content.
* [[[inc:cleanup]]] for a page that needs to be cleaned up.
* [[[inc:cid]]] for a page that is currently in development
 * extra = extra information that will be displayed;
 * You probably don't need to worry about the [[[inc:cid-backend]]].
* [[[inc:development]]] for a page that is a member of the development cycle.
