# Mod

This command returns the modulo of two numbers. This is defined as the remainder when *value1* is divided by *value2*. This command is identical to the [nspire:remain](nspire:remain.html) command, which returns the remainder of the dividend. 

```
mod(12,5) = 2

mod(8,2) = 0

mod(64, 15) = 4
```
